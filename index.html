Block & Inline?

In HTML, Block element always begins on the new line, it extends from left to right where browser automatically gives some space(margin) from both the side. Inline Element does not begin in new line, it only takes the neccessary width required for it. Examples Block : div, h1 to h6, table, p, form etc. Examples Inline : span, img, a etc.
